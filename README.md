📌 Executive Summary

This project implements a Neural Language Model using Long Short-Term Memory (LSTM) networks in PyTorch to perform Next Word Prediction.

The model learns sequential word dependencies from a structured FAQ-based corpus and generates context-aware text predictions.

It demonstrates strong understanding of:

Sequence Modeling

Language Modeling

Tokenization & Vocabulary Engineering

Custom Dataset Pipelines

Deep Learning Training Loops

Text Generation Mechanisms

Model Evaluation Metrics
